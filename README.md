# Project-shootBubble
原型： https://org.modao.cc/app/b50d567f62006de9f462ffa1d8f165fb 
玩法： 界面上分布着9个气球，以3 X 3的方式排列。用户在X和Y上选择1-3的数字来定位气球，用户点击“发射”后，对应坐标位置的气球被击破，当界面上所有气球都被击破后，获得胜利 
规则： 1.总共分布3 X 3共9个气球 
       2.被击破的气球在当局游戏中不会再出现 
       3.所有气球被击破获得游戏胜利 
       4.已被击破的气球会在当局游戏中消失 
       5.可通过UI上“重新开始”按钮重置游戏
